# qt qml cmake template
qt qml cmake template for scaffolding/bootstrapping new project

### directory structure

```sh
tree -d

.
├── 3rd_party
├── app
│   ├── qml
│   ├── resources
│   │   ├── audios
│   │   ├── fonts
│   │   │   ├── Roboto
│   │   │   └── SFPro
│   │   │       ├── OpenType
│   │   │       └── TrueType
│   │   ├── images
│   │   └── styles
│   └── src
└── app_libs
```
